Sequence by month, day, year

Key features:
  • Lastest update: v15.21.11.23
  • Sequence Customize the sequence interval.
  • Sequence reset by day, reset by month, reset by year
  • Multi language support.

Sample

  • In April 2019, sale order: SO/20190401/001, SO/20190401/002, SO/20190401/003.
  • In May 2019, sale order: SO/20190501/001, SO/20190501/002, SO/20190501/003.

How to use

  • Enable development mode.
  • Go Technical->Sequences & Identifiers -> Sequences, to set
  • The date is base on the creator's timezone. If you set the Frequency, the date range and Next Number would create auto. Also you can customize From/to/Next Number.
  • Once you change the Frequency, you must remove the exist date range under to reset.
